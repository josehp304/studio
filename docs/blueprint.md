# **App Name**: Dream LMS Clone

## Core Features:

- Course Listing: Display a list of courses with key information such as title, instructor, and brief description based on the provided url: https://dreamslms.dreamstechnologies.com/react/template/course/course-list
- Course Details Page: Replicate the course details page, showcasing comprehensive course information.
- User Authentication: Implement user login and registration functionalities to manage access to courses.
- Responsive Design: Ensure the LMS template is fully responsive and adapts to different screen sizes and devices.
- Search and Filter: Enable users to search and filter courses based on keywords, categories, or instructor.
- UI components: Reproduce buttons, forms, icons, and navigation to completely mirror the Dream LMS template.

## Style Guidelines:

- Primary color: Deep blue (#191762) to convey trust, security, and intelligence, echoing the nature of education.
- Background color: Very light gray (#F0F2F5), close in hue to the primary color, for a clean and professional look.
- Accent color: Soft purple (#9370DB) for highlights and interactive elements, an analogous color choice that is calming.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, and 'Inter' (sans-serif) for body text, to establish a clear typographic hierarchy that still maintains a futuristic, fresh style.
- Use a consistent style of modern and minimalist icons to represent course categories and actions.
- Follow a clean and structured layout with clear sections for course listings, details, and navigation. Copy the layout of https://dreamslms.dreamstechnologies.com/react/template/course/course-list exactly.
- Implement subtle transitions and animations to enhance user experience and provide visual feedback, but use animation sparingly.
